# Movie-recommendation-Final
# 🎬 Movie Recommendation System

This project is a simple movie recommendation system built using Python and Streamlit.

## Dataset

The dataset used in this project is too large to upload to GitHub.

You can download it here:
[https://www.kaggle.com/datasets/rounakbanik/the-movies-dataset](https://www.kaggle.com/datasets/rounakbanik/the-movies-dataset)

After downloading, place `movies_metadata.csv` in the project folder.

## Features
- Genre-based movie recommendations
- Minimum rating filter
- Clean and simple interface

## Technologies
- Python
- Pandas
- Streamlit

## Learning
This project was inspired by:
- AI Python for Beginners
- Become a Data Storyteller with Streamlit

## How to Run

1. Install dependencies:
pip install pandas streamlit

2. Run the app:
streamlit run app.py

## Iteration Process

- Iteration 1:
  Raw data used → noisy and unclear movie titles

- Iteration 2:
  System worked but still poor usability

- Iteration 3:
  Switched to genre-based recommendations → improved usability and stability
  
It focuses on experimentation, iteration, and usability.
